Coffee Sales Dashboard

📜 Project Overview

The Coffee Sales Dashboard project is an Excel-based interactive tool designed to analyze and visualize sales data for coffee products. This dashboard provides stakeholders with actionable insights into total sales, trends, top customers, and sales distribution by country, enabling informed decision-making.

🚀 Key Features

Interactive Filters:

Filters for Date, Roast Type (Dark, Light, Medium), Size (0.2 kg, 0.5 kg, 1.0 kg, 2.5 kg), and Loyalty Card (Yes/No).

Period selector for yearly and monthly analysis.

Visualizations:

Total Sales Over Time: A line chart displaying sales trends over multiple years.

Sales by Country: A bar chart comparing sales in various countries (e.g., United States, Ireland, United Kingdom).

Top 5 Customers: A bar chart listing the top customers based on sales.

Track Attributes: Detailed track characteristics such as energy, danceability, and more.

Tabs:

Dashboard: The main interactive dashboard.

TotalSales: Detailed total sales data.

CountryBarChart: Sales distribution by country.

Top5Customers: Data for the top customers.

Orders, Customers, Products: Supporting datasets.

Customization:

Easily adaptable filters and charts to meet specific business requirements.

📊 Dashboard Highlights

📖 How to Use

Download the coffeeOrdersProject.xlsx file from this repository.

Open it using Microsoft Excel (2016 or later).

Navigate to the Dashboard tab to explore the visualizations.

Interact with the filters and slicers to uncover actionable insights.
